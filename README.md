# Proyecto Integrador: Desarrollo de Sistemas Orientado a Objetos (DSOO)

## 📌 Información General
* **Materia:** Desarrollo de Sistemas Orientado a Objetos
* **Carrera:** Tecnicatura Superior en Desarrollo de Software (IFTS N° 29)
* **Cuatrimestre:** 1° Cuatrimestre 2026
* **Lenguaje:** C# (.NET)

## 🎯 Objetivos del Proyecto
Basado en el programa oficial de la materia, este proyecto busca:
1.  **Aplicar el Paradigma Orientado a Objetos (POO):** Implementar conceptos de abstracción, encapsulamiento, modularización y jerarquía de clases.
2.  **Modelado UML:** Diseñar la arquitectura del sistema mediante diagramas de clases y casos de uso antes de la codificación.
3.  **Calidad de Software:** Aplicar buenas prácticas de programación, normas de calidad y manejo de excepciones.
4.  **Documentación Técnica:** Elaborar el manual de usuario y la documentación técnica correspondiente.

## 📝 Consigna del Trabajo
Desarrollar una aplicación de consola (o entorno visual según avance el curso) que permita resolver una problemática de gestión de datos, cumpliendo con los siguientes requisitos:

### 1. Estructuras Fundamentales
- Uso de **variables y tipos de datos** adecuados (int, double, string, bool).
- Implementación de **estructuras de control**:
  - Condicionales (`if`, `else if`, `switch`).
  - Ciclos (`for`, `while`, `foreach`).
- Entrada y salida de datos mediante `Console.ReadLine()` y `Console.WriteLine()`.

### 2. Lógica de Negocio (POO)
- Creación de **Clases** representativas del dominio del problema.
- Instanciación de **Objetos** y comunicación entre ellos a través de métodos.
- Uso de **Constructores** y propiedades (Getters/Setters).

### 3. Validación de Datos
- Implementación de métodos de conversión como `int.TryParse()` o `double.Parse()` para asegurar la integridad de la entrada del usuario.
